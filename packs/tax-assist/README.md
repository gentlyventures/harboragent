# üìã Key 2025 Changes Covered

## Individual Tax Changes
- Inflation adjustments (brackets, standard deduction, AMT thresholds)
- Standard mileage rates for 2025
- Retirement contribution limits (401(k), IRA, SIMPLE, catch-up)
- ACA premium tax credit parameters
- New Form 1099-DA (Digital Asset Broker Reporting)
- Form 7203 updates (S Corporation Shareholder Basis)
- Forms 8995/8995-A (QBI deduction) updates

## Corporate & Pass-Through Changes
- Expanded e-filing requirements for businesses
- Pass-through entity (PTE) tax clarifications and SALT cap workarounds
- K-1 schedule updates (Forms 1065, 1120-S)
- State PTE tax regime federal treatment

## Software & Technology Changes
- Modernized e-File (MeF) program updates and schemas
- IRS Direct File expansion
- Digital asset reporting requirements
- New form structures and validation rules

## AI Workflow Opportunities
- Document processing automation
- Data extraction from 1099s, W-2s, K-1s
- Quality assurance and error detection
- Client communication automation
